## Installation
Via [npm] (https://www.npmjs.com/):
```bash
npm i --save accounting-diary
```
Via [yarn] (https://yarnpkg.com/):

```bash
yarn add accounting-diary
```
In your code:
 ````jsx
import React from "react";
import * as AccountingDiary from "accounting-diary";

function App() {
  return (
    <div>
      <AccountingDiary/>
    </div>);
}

export default App;

 ````
